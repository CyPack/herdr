# Herdr Native-FM Edge Cases

| Situation | Solution | Date |
|-----------|----------|------|
| A warned MCP has no direct `mcp_servers` entry in `config.toml` | Inspect enabled plugin declarations and the matching plugin MCP manifest; plugins can inject named MCP servers such as GitHub | 2026-07-14 |
| A `Type=simple` MCP proxy initializes every named stdio backend serially before opening its listener | Size readiness from a measured isolated cold start plus bounded headroom; keep exact server-set and critical MCP probes so a larger timeout cannot turn a genuine hang into a false green | 2026-07-14 |
| A throwaway terminal host window exists while a Rust probe is still compiling, so an early screenshot captures startup text rather than graphics evidence | Pre-build the ignored probe or wait for its explicit on-screen marker before capture; inspect the capture and reject compile/startup-only images | 2026-07-14 |
| A Ratatui buffer test derives a cell coordinate with `str::find` from a row containing Unicode separators | Convert the byte offset to a character count before indexing terminal cells; UTF-8 byte positions are not screen-cell positions | 2026-07-14 |
| An isolated `--no-session` PTY launched from inside Herdr still inherits session identity after socket overrides are cleared | Unset `HERDR_ENV`, `HERDR_WORKSPACE_ID`, `HERDR_TAB_ID`, and `HERDR_PANE_ID` together with both socket variables; use unique throwaway XDG roots and verify semantic exit plus zero residue | 2026-07-14 |
| Several individually valid fixed-width row buttons share a narrow responsive Miller CURRENT column | Run ordinary-name readability characterizations in addition to geometry tests; keep every target complete and disjoint, but compact or progressively hide controls before they consume the usable name identity width | 2026-07-14 |
| A file-manager row hit area snapshots only an entry index while watcher refresh can reorder the visible list | Snapshot stable path identity with the row geometry and require the live entry at that index to match before any plain, Ctrl, or Shift selection mutation; consume stale same-index hits without changing cursor or selection | 2026-07-14 |
| A Ratatui `TestBackend` assertion inspects every display cell covered by a double-width Unicode glyph | Assert the style on each glyph-start cell and skip the backend's reset blank continuation cells; those hidden cells are an internal buffer representation, not a visible background hole | 2026-07-15 |
| A frozen full-frame wall-clock budget fails or varies widely in the default debug test profile while the optimized product path is healthy | Keep default nextest deterministic by running a small workload plus structural invariants; enforce the unchanged p95 budgets in an explicit `cargo nextest run --release` gate with warm-up, fixed sample count, and printed measurements | 2026-07-17 |
| A test-owned tmux command passed as one quoted command string exits before the client becomes observable | Use a test-owned interactive shell as the tmux command, send the fully explicit environment-clearing client command into that shell, preserve stderr for diagnosis, then close with Herdr semantic detach followed by shell `exit` | 2026-07-17 |
| One physical high-resolution or momentum wheel gesture may decode into several terminal `ScrollUp`/`ScrollDown` events | Measure physical gesture, raw decoded-event count, dispatcher count, and reducer-mutation count separately; first prove one decoded unit moves one owner-column row, then authorize normalization only at the stage where multiplicity is observed | 2026-07-21 |
| A wheel/key burst lands on a directory and later events appear to skip several rows | Automatic directory branching can transfer the active column after the first event, so the remaining burst mutates the child and masquerades as a pure wheel-rate problem | Decouple cursor movement from activation before tuning wheel cadence; retain the original owner column through the burst and rerun the physical trace | 2026-07-21 |
| A user says the original stutter is gone while no new numeric profiler run is attached to that exact statement | Treat it as high-value qualitative symptom acceptance but keep it distinct from structural tests, prior profiler counts, and fresh publication gates; record evidence type and date rather than inventing a metric | 2026-07-21 |
| Yazi demonstrates fast directory reuse through `History(HashMap<UrlBuf, Folder>)` but exposes no memory cap or eviction in the cited implementation | Transfer its async cancellation, ticket, cursor/activation separation, and change-gated rendering; do not treat the unbounded history map as authorization for a Herdr cache | 2026-07-21 |
| Ghostty emits identical-direction and identical-coordinate vertical SGR packets in triplets or occasional sextuplets less than 2 ms apart | Coalesce only the same Files generation, Trail owner, direction, and coordinates strictly below 2 ms; advance the stamp on every packet and preserve reversal, owner/coordinate changes, the 2 ms boundary, and later detents | 2026-07-21 |
| A cursor-only directory move temporarily disagrees with the previously activated child branch while async preview is pending | Keep the resident chain in state for rollback/stale safety, highlight the exact cursor, and hide only the mismatched child from projection until a matching bounded preview arrives | 2026-07-21 |
| `TrailState::deepest()` is greater than `active_col()` because a child preview is resident but not entered | Treat `deepest()` as prepared-data extent only; retain the snapshot, but use `active_col()` for input, auto-follow, render, hit testing, resize, and watcher ownership | 2026-07-22 |
| Explicit Leave/Backspace returns to a parent while the former child snapshot remains resident | Rebind the watcher exactly once to the parent owner and require the following sync to be inert; do not infer watcher ownership from resident depth | 2026-07-22 |
| A deterministic visual tree contains symlinks, FIFOs, async previews, and mtime-group sorting | Set link/FIFO timestamps without following targets, use equal explicit mtimes where name order is the contract, inject the calendar clock, and wait for exact preview identity/generation before export | 2026-07-22 |
| A file cursor is active in a parent column while an older directory child remains resident to the right | Resident availability does not authorize Right to enter or activate the file; exact cursor kind wins, so Right/`l` is inert and the prepared child remains unchanged | 2026-07-22 |
